# Eisher Industries LLP

Premium multi-page website for Eisher Industries LLP — Nature Friend.

## Stack

- Next.js 15 App Router
- TypeScript and Tailwind CSS
- Motion, GSAP and Lenis
- React Three Fiber and Three.js

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run check
npm run build
```

The project is configured for deployment on Vercel with no required environment variables.
